Search Window
=============

The Search window provides advanced filtering to locate specific patients within the database.

Overview
--------

Patients can be filtered by their clinical information, gender, and examination date. The search logic can be set to:

- **AND**: All conditions must be satisfied
- **OR**: At least one condition must be satisfied

Exporting Results
-----------------

The list of filtered subjects (or the entire database) can be exported to the Export window for further analysis. This provides an overview of the existing subjects and is essential for managing large databases.
