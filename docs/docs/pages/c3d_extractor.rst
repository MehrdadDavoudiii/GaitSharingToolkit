C3D Extractor
=============

The C3D Extractor converts biomechanical data from the C3D format into readable Excel spreadsheets.

.. image:: /_static/images/c3d_extractor.png
   :width: 100%
   :alt: C3D Extractor
   :align: center

|

Overview
--------

C3D is the standard format for saving the outputs of clinical gait analysis. However, working with this type of file is challenging, especially for clinicians. The C3D Extractor provides the ability to extract the required data from one or multiple C3D files into an Excel spreadsheet.

Output
------

.. image:: /_static/images/c3d_output.png
   :width: 100%
   :alt: C3D Extractor Output
   :align: center

|

The output (``.xlsx`` format) contains the results for the entire gait cycle, extracted from the data saved in the C3D file. This includes kinematics (joint angles), kinetics (moments and powers), and any other recorded parameters.
