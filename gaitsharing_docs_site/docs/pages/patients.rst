Patients Window
===============

The Patients window displays all imported patient records in a searchable table view.

.. image:: /_static/images/patients_window.png
   :width: 100%
   :alt: Patients Window
   :align: center

|

Overview
--------

The imported patients' records, including name, last name, diagnosis, examination date, and other clinical details, can be viewed in this window.

Editing Records
---------------

Records can be edited or deleted by **double-clicking** on them. This opens an inline editor where fields can be modified directly.

.. note::
   Deleted records are soft-deleted (archived) by default. This prevents accidental data loss and blocks duplicate re-imports of the same patient.
