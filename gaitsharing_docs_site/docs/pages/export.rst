Export Window
=============

The Export window enables creating secondary databases for research and analysis.

.. image:: /_static/images/export_window.png
   :width: 100%
   :alt: Export Window
   :align: center

|

Overview
--------

This option helps create a secondary database containing only specific files (e.g., C3D for research purposes) for the filtered patients. This secondary database avoids applying changes to the original patient data, especially when planning further processing.

The location for this collection can be selected through the **Browse** button.

Export Summary
--------------

A summary Excel file (``export_summary.xlsx``) will also be generated during the export process, providing a tabular overview of all exported subjects and their metadata.
